# RucksackBot

## Start

1. Node.js installieren
2. Ordner herunterladen
3. `npm install` ausführen
4. `config.example.json` kopieren und in `config.json` umbenennen
5. Twitch-Name und OAuth-Token eintragen
6. `start.bat` doppelklicken

## Befehle

!wind  
!eis

!queue
